This repository contains the source code for generating figures and analyses described in the manuscript:

> **[ Aberrant neuron-OPC synaptic transmission and deficient myelination define sex bias in major depressive disorder]**

All code is written in **R** (≥ 4.2). Raw snRNA‑seq data are available from GEO (GSE144136, GSE213982). Processed intermediate files (e.g., CellChat objects) are not included due to size; they can be regenerated by running the preprocessing scripts.
